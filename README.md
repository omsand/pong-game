# Pong Game 🏓

A browser-based Pong game built with TypeScript.

## Demo

Play here:

https://pong-game-three-tau.vercel.app

## Features

- Player paddle controls
- Computer opponent
- Ball physics
- Wall collisions
- Paddle collision detection
- Score tracking
- Win / lose states
- Restart functionality

## Tech Stack

- TypeScript
- Vite
- HTML Canvas
- Git + GitHub
- Vercel Deployment

## What I Learned

This project was built to practice real software development workflows:

- Setting up a TypeScript project with the support of AI
- Working with the HTML Canvas API
- Managing game state
- Creating a game loop
- Debugging TypeScript errors
- Using feature branches
- Creating pull requests
- Deploying production builds
